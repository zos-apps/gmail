## About

Email client with full Gmail integration.

## Features

- Email composition
- Labels
- Search
- Filters
- Attachments

## Installation

### From App Store
Open the **App Store** in zOS and search for "Gmail".

### Via npm
```bash
npm install @zos-apps/gmail
```

## Usage

- Click Compose to write
- Use labels to organize
- Search to find emails



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
